Olaylar (Event) Referansı

Yazanlar Danielle Kain ve Kevin Miller

Öncelikle ne zaman MouseEvent kullanırsanız aşağıdaki satırı run metodunuza eklemeyi unutmayın:
addMouseListeners();
Böylece programa MouseEvent'lere dikkat etmesi gerektiğini söylersiniz. KeyListeners kullanıyorsanız da benzeri şekilde aşağıdaki satırı run metodunuza ekleyin:
addKeyListeners();
Bu şekilde listener'larınızı kurduktan sonra, sıra onları kodda kullanmaya geliyor. Aşağıda Breakout için işinize yarayabilecek dört olayı listeledik. Tam bir liste isterseniz KeyEvent'ler için bu linke, MouseEvent'ler içinse bu linke bakabilirsiniz.

public void mouseClicked(MouseEvent e) {
	//Mouse her tıklandığında bu metod çağrılır.
}

public void mouseMoved(MouseEvent e) {
	//Mouse her hareket ettiğinde bu metod çağrılır.
}
			
public void keyPressed(KeyEvent e) {
	//Mouse'a her basıldığında bu metod çağrılır.
}

waitForClick(); //Bu satır kullanıcı mouse'a tıklayana kadar programı durdurur.

Aşağıda bazı olay kullanım örneklerini görebilirsiniz: 

public void mouseClicked(MouseEvent e) {
	System.out.println("x: " + e.getX() + " y: " + e.getY());
	//Bu metod tıkladığınız noktanın koordinatlarını konsola yazar.
}

public void keyPressed(KeyEvent e) {
	System.out.println("key pressed: " + e.getKey());
	//Bu metod klavyede bastığınız tuşu konsola yazar.
}
